# 人生経営シミュレーション（Life as Victoria 3）

資格勉強とは別に、**日々の生活・キャリア構築を Victoria 3 の国家運営になぞらえてゲーミフィケーション化する**取り組み。
「自分＝ひとつの国家」と見立て、4つの省庁（生産・政治・外交・軍事）にタスクを割り当て、行動の積み重ねを国力（ステータス）の成長として可視化する。

- 目的: 継続しづらい生活・キャリア行動に、Vic3的な「積み上げの実感」と「相互作用」を与える
- スコープ: 資格（takken/nw/sc）の外側にある、生活全般・心身・キャリアの取り組み
- Vic3対応: 4省庁 ＝ 生産（経済）／政治（内政）／外交／軍事

---

## 国家ステータス（共通指標）

各タスクは以下の国力指標のいずれかを上昇させる。Vic3のステータスに対応。

| 指標 | 記号 | 意味（人生換算） | Vic3対応 |
|---|---|---|---|
| 技術力 | 🔬 | 知識・スキル・専門性 | 研究力 / イノベーション |
| 体力 | 💪 | 健康・持久力・見た目 | 労働力の質 / 人口 |
| 生活水準 | 🏠 | 暮らしの快適さ・秩序・余裕 | SoL（生活水準） |
| 資金 | 💰 | 現金・貯蓄・収入 | GDP / 国庫 |
| 権威 | ⚙️ | 自己規律・実行力・時間の可処分性 | 権威（Authority） |
| 威信 | 🏛️ | 対外的な評判・存在感 | 威信（Prestige） |
| 影響力 | 🤝 | 人脈・コミュニティでの発言力 | 影響力（Influence） |
| 交渉力 | ⚔️ | 条件を勝ち取る力・市場価値 | 軍事力 / 士気 |

> **相互作用の例**: 生産で💪体力と🔬技術力を貯める → 外交で🤝影響力に変換 → 軍事（転職交渉）で💰資金と⚔️交渉力に結実。政治（家事）で🏠生活水準と⚙️権威を維持できないと、他省庁の行動が滞る（＝Vic3の「権威が尽きると法令を通せない」構造）。

## 🏭 生産（Production）— 心身の能力向上

> Vic3対応: 経済・建築。国の生産力そのものを底上げする最重要の内政。

個人の心身の能力を高める、すべての土台となる投資。ここへの投資が他省庁の行動力の源になる。

| タスク | 具体例 | 頻度目安 | 効果 |
|---|---|---|---|
| 学習（資格外） | 技術書・語学・投資/簿記の勉強、写経 | 毎日 | 🔬技術力 +大 / ⚙️権威 +小 |
| 筋トレ | 自重トレ・ジム・ランニング | 週3〜 | 💪体力 +大 |
| 有酸素・活動 | ウォーキング、階段、通勤の運動化 | 毎日 | 💪体力 +小 / 🏠生活水準 +小 |
| 睡眠・回復 | 7時間睡眠、就寝時刻の固定 | 毎日 | ⚙️権威 +中 / 💪体力 +中 |
| 食事管理 | 自炊・タンパク質確保・節酒 | 毎日 | 💪体力 +中 / 💰資金 +小 |
| インプット | 読書、良質な記事/動画の消化 | 毎日 | 🔬技術力 +中 |

**建設ラッシュ（集中投資）**: 一定期間ひとつの能力に資源を集中する（例: 3ヶ月の筋トレ強化期）。Vic3の「建設セクター拡張」に相当し、短期は他が停滞するが長期の生産力が跳ねる。

## 🏛️ 政治（Politics）— 生活維持のタスク

> Vic3対応: 内政・法令・権威。国を安定稼働させる土台。ここが崩れると全省庁が機能不全になる。

生活を回すための家事・雑務。地味だが、これを自動化・習慣化して⚙️権威（可処分の実行力）を確保できるかが全体の生産性を決める。

| タスク | 具体例 | 頻度目安 | 効果 |
|---|---|---|---|
| 掃除・整理 | 部屋の片付け、水回り掃除 | 週次 | 🏠生活水準 +中 / ⚙️権威 +小 |
| 洗濯・衣類管理 | 洗濯、アイロン、衣替え | 週次 | 🏠生活水準 +小 |
| 炊事 | 献立・買い出し・作り置き | 毎日/週次 | 🏠生活水準 +中 / 💪体力 +小 |
| 金銭管理 | 家計簿、固定費見直し、資産管理 | 週/月次 | 💰資金 +中 / ⚙️権威 +小 |
| 事務手続き | 支払い、行政/契約手続き、通院 | 随時 | 🏠生活水準 +小 / ⚙️権威 +中 |
| 環境整備・自動化 | 定位置化、サブスク/家電/仕組み化 | 単発 | ⚙️権威 +大（恒久） |

**法令（習慣化）**: タスクを「毎週◯曜に掃除」のようにルール化＝Vic3の法令制定。制定時は⚙️権威を消費するが、通ると恒久的に生活水準が底上げされ、以後の手間が減る。

## 🕊️ 外交（Diplomacy）— キャリアにつながる対外活動

> Vic3対応: 外交・関係構築。他国（他者・コミュニティ）との関係を築き、影響力と威信を得る。

新たなキャリアの種をまく、対外的な取り組み。すぐ資金にはならないが、🤝影響力と🏛️威信を蓄積し、軍事（交渉）の成功率を上げる下地になる。

| タスク | 具体例 | 頻度目安 | 効果 |
|---|---|---|---|
| 勉強会・イベント参加 | 技術/業界の勉強会、カンファレンス | 月次 | 🤝影響力 +中 / 🔬技術力 +小 |
| コミュニティ運営 | 主催・登壇・モデレート | 随時 | 🏛️威信 +大 / 🤝影響力 +中 |
| 情報発信 | ブログ、登壇資料、SNS/OSS | 週/月次 | 🏛️威信 +中 / 🤝影響力 +中 |
| 副業・案件（種まき） | 小規模受託、個人開発の公開 | 随時 | 💰資金 +小 / 🔬技術力 +中 / 🏛️威信 +小 |
| サークル・課外活動 | 趣味サークル、社外の横のつながり | 随時 | 🤝影響力 +小 / 🏠生活水準 +小 |
| 人脈メンテ | 元同僚/知人との定期連絡、1on1 | 月次 | 🤝影響力 +中 |

**同盟・関係改善**: 継続的な発信・参加で特定コミュニティとの関係度が上がる（Vic3の関係度）。関係度が高いほど、後の軍事フェーズ（転職/案件交渉）でリファラルや推薦という「参戦」が得られる。

## ⚔️ 軍事（Military）— 交渉・条件獲得

> Vic3対応: 軍事・戦争。蓄えた国力を「実際の利益」に変換する、決着のフェーズ。

転職活動・案件獲得など、**条件を勝ち取る交渉**。生産（実力）・政治（生活の安定）・外交（人脈と威信）で蓄えた国力を、💰資金と⚔️交渉力（市場価値）に換金する。

| タスク | 具体例 | 頻度目安 | 効果 |
|---|---|---|---|
| 装備更新 | 職務経歴書・ポートフォリオ整備 | 事前 | ⚔️交渉力 +中（準備） |
| 偵察 | 求人/相場リサーチ、市場価値の把握 | 事前 | ⚔️交渉力 +小 / 情報 |
| 演習 | 面接練習、模擬交渉、想定問答 | 事前 | ⚔️交渉力 +中 |
| 開戦（応募・提案） | 転職応募、案件提案、見積提示 | 随時 | 進行度 +（威信/影響力で成功率補正） |
| 交渉（会戦） | 面接、条件交渉、単価交渉 | 随時 | 💰資金 +大 / ⚔️交渉力 +大 |
| 講和（合意） | オファー承諾、契約締結 | 決着時 | 💰資金 +大（恒久） / 🏛️威信 +中 |

**戦争目標（ウォーゴール）**: 開戦前に「年収+◯%」「リモート可」「裁量◯」など獲得条件を明確化。外交で貯めた🏛️威信・🤝影響力が高いほど、譲歩を引き出しやすい＝有利な講和条件になる。準備（生産・外交）なき開戦は消耗戦になりやすい。

## 省庁間の循環（ゲームループ）

```
  🏭生産 ──(実力・健康)──▶ 🕊️外交 ──(人脈・威信)──▶ ⚔️軍事
    ▲                                                    │
    │                                              (資金・地位)
    └──────────── 🏛️政治（生活の安定・権威）◀───────────┘
         生活が回るほど、各省庁に投じられる行動力(権威)が増える
```

- **政治が土台**: 生活が破綻すると権威が尽き、他のすべてが動かせなくなる
- **生産が国力の源泉**: 実力と健康なくして外交・軍事は空回りする
- **外交は軍事の前提**: 人脈・評判の蓄積が、交渉の成功率と条件を左右する
- **軍事で回収**: 蓄えた国力を資金・地位に換金し、次の生産投資に回す

## スコアリング（行動 → 国力の計算）

各タスクの `+小/+中/+大` は行動値（AP: Action Point）に換算する。ステータスは累積し、レベルに変換される。

| 表記 | 行動値 |
|---|---|
| +小 | 1 AP |
| +中 | 3 AP |
| +大 | 5 AP |

- **累積値**: ステータスごとにAPを積み上げる（減らない資産）。
- **レベル**: `レベル = floor(√(累積AP ÷ 10))`。序盤は伸びやすく、上級で鈍化する（Vic3の逓減）。
  - 例: 累積40 → Lv2、160 → Lv4、360 → Lv6、640 → Lv8。
- **勢い（Momentum）**: 直近4週の週平均AP。Vic3の「建設速度」に相当し、継続すると次項のイベント好機が増える。**2週連続でそのステータスのAPが0だと減衰**（-1 AP/週、Vic3の減価）。継続にインセンティブを与えるための唯一のマイナス要素。

> 単発の大きな行動より、**毎週コンスタントに積むほうがレベルが伸びる**設計。三日坊主は勢いが減衰し、習慣は複利で効く。

---

## 国家ランク（総合国力の段位）

`総合国力 = 8ステータスのレベル合計`。Vic3の列強序列になぞらえた段位を付与する。

| 段位 | 総合国力 | 状態イメージ |
|---|---|---|
| 未承認国家 | 0–7 | 立ち上げ期。まず生活と生産の土台づくり |
| 弱小国 | 8–15 | ルーティンが回り始めた |
| 地域国 | 16–25 | 得意分野（尖ったステータス）が出てきた |
| 主要国 | 26–39 | 複数分野が成熟。外交・軍事で成果が出始める |
| 列強 | 40–55 | 全方位で高水準。キャリアの選択肢が広い |
| 覇権国 | 56+ | 突出。目標を自分で設計するフェーズ |

**バランス補正**: 最も低いステータスのレベルが全体平均の半分未満だと「**不安定**」判定（Vic3の急進派増大）。段位表示に⚠️が付き、次項の危機イベント発生率が上がる。尖りだけでなく穴を埋める動機づけ。

## 利益団体（心の中の勢力）

自分の中には複数の欲求＝利益団体（IG: Interest Group）が同居し、それぞれ求める投資先が違う。**満足度**を保つと国が安定し、無視すると急進化してモチベーション低下・破綻を招く。Vic3の内政の核。

| 利益団体 | 求めるもの（投資先） | 満足時のボーナス | 急進化（無視）のリスク |
|---|---|---|---|
| 🔬 向上派（知識人） | 生産（学習・成長） | ⚙️権威 +（実行力が上がる） | 停滞感・自己肯定感の低下 |
| 💰 実利派（実業家） | 軍事・外交（収入・キャリア） | 💰資金の獲得効率 + | 焦り・現状への不満 |
| 🏠 安寧派（小市民） | 政治（生活の安定・休息） | 生活水準の維持コスト減 | 生活の乱れ・体調不良 |
| 🤝 社交派 | 外交（人脈・コミュニティ） | 🏛️威信・🤝影響力 + | 孤立・情報の遅れ |
| 🎮 享楽派（民衆） | 余暇・娯楽・休養 | 全ステータスのAP +5%（士気） | **反乱＝燃え尽き**。全行動力が急減 |

**運用ルール**:
- 各IGの満足度は「直近2週にその領域へ投資したAP」で判定（0なら不満、一定以上で満足）。
- 享楽派を軽視した「ブラック統治」は短期に生産が伸びるが、**燃え尽きイベント（下記）で全停止**するハイリスク運用。意図的に休養日を「投資」として扱うこと。
- 満足度が対立することもある（実利派＝働け／安寧派＝休め）。どちらを優先するかが各週の「政策判断」。

## ジャーナル（四半期の戦争目標）

Vic3のジャーナル項目になぞらえ、**四半期ごとに1〜3個の期限付き目標**を立てる。各目標に達成条件と報酬（ボーナスAP）を設定。

**フォーマット例**:

| 目標 | 省庁 | 達成条件 | 期限 | 報酬 |
|---|---|---|---|---|
| 体力の建設ラッシュ | 🏭生産 | 週3トレを12週継続 | Q末 | 💪体力 +30 AP 一括 |
| コミュニティで登壇 | 🕊️外交 | 勉強会で1回発表 | Q末 | 🏛️威信 +25 / 🤝影響力 +15 |
| 転職の開戦準備 | ⚔️軍事 | 職務経歴書＋応募5件 | Q末 | ⚔️交渉力 +20 |

- **期限切れ**: 達成できなくても罰則なし。ただし「未完の懸案」として翌Qに繰り越し、放置し続けると安寧派/実利派の不満要因になる。
- **達成報酬**は一括AP。日々のコツコツ（生産）に対し、ジャーナルは中期の大きな跳躍を担う。

## 週次サイクル（ターンの回し方）

Vic3の時間進行になぞらえ、**1週 = 1ターン**。日曜を決算日とする。

1. **決算（日曜）**: 1週分のタスクをAPに集計 → 各ステータスに加算 → レベル・段位・IG満足度を更新。
2. **勢いチェック**: 各ステータスの週平均を見て、減衰対象（2週0AP）を確認。
3. **予算配分（＝権威の割り当て）**: 翌週にどの省庁へ注力するか、⚙️権威レベルを上限に配分を決める。権威が高いほど多くの省庁を同時進行できる（Vic3の「権威が法令数の上限」構造）。
4. **ジャーナル確認**: 四半期目標の進捗を1行更新。
5. **イベント判定**: 下記テーブルで今週の好機/危機をチェック。

---

## イベント表（好機と危機）

各週の決算時、条件を満たすと発火。Vic3のイベント/危機に相当し、単調なスコア積みに揺らぎを与える。

**🟢 好機（条件達成でボーナス）**

| イベント | 発火条件 | 効果 |
|---|---|---|
| 技術的ブレイクスルー | 🔬技術力の勢いが高い週 | 学習AP +50%（1週） |
| 好意的な評判 | 🏛️威信レベル ≥ 5 | 外交タスクの成功に🤝影響力ボーナス |
| 追い風の市況 | ⚔️軍事フェーズ中に威信が高い | 交渉の獲得資金 +（有利な講和条件） |
| 高い士気 | 享楽派が満足 | 全ステータスAP +5%（1週） |

**🔴 危機（放置のツケ）**

| イベント | 発火条件 | 効果 |
|---|---|---|
| 燃え尽き（反乱） | 享楽派の急進化が継続 | 全行動力ダウン。1〜2週は休養を強制、生産・軍事の新規行動不可 |
| 生活の破綻 | 🏠生活水準が2週連続で低下 | ⚙️権威が減り、同時進行できる省庁数が減少 |
| 市場価値の陳腐化 | 🔬技術力が長期0AP | ⚔️交渉力レベルにマイナス補正 |
| 孤立 | 社交派が長期不満 | 外交の好機イベントが発生しなくなる |
